Sobre el proyecto

Este proyecto fue hecho con el propósito de representar visualmente una página web del clima. Los datos mostrados son simulados y no reflejan condiciones reales.

Tecnologías usadas

Servidor

Node.js

Estilos

Bootstrap 5

Mapas

Windy Embed

Frontend

HTML / CSS / JS

Fuentes de datos

Los datos de temperatura, humedad, presión, viento e índice UV son generados de forma aleatoria mediante funciones que utilizan Math.random(). No representan condiciones meteorológicas reales.

Créditos

  • Windy.com Mapa climático interactivo
  • Bootstrap Componentes y estilos
